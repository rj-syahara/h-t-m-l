# このプログラムについて
この診断プログラムは、N予備校 Webプログラミング入門コース「あなたのいいところ診断」をベースに、学習した成果を盛り込んで改造したものです。

## 予定
- 名前を入力 -> 名前を数字化して、いいところを診断
↓
- 名前と生年月日を入力 -> 今日の日付＋名前の数値化＋生年月日で、ラッキーアイテムを診断
